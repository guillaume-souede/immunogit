Search Tips/Tricks

  • Search is based on exact word matches. Make sure your spelling is correct. Example: the misspelled terms like CelCycle (missing "l") will not find any results.
  • Search for the phrase "Glucose" in the name field. Example: name:"Glucose"
  • Matches all models having model elements that were annotated with Gene Ontology term cell growth. Example: GO:GO:0016049

What are BioModels domains?

This drop down box allows you to filter which models appear in the search results:

  • Autogenerated Models: large-scale or automatically-generated submissions such as Path2Models or Patient-derived genome scale metabolic models (PDGSMM)
  • Individual Models: models described in the literature that have been submitted individually to BioModels
  • All Models: the whole BioModels content

The default search domain is Individual Models.
